Summary of what we have learned today, 26th August 2020:

1.	Mengerti tentang beberapa perintah Gitlab yang dijalankan melalui terminal (bash) seperti: init, add, push, pull, clone
2.	Mengetahui bagaimana cara create project di Gitlab

a.	Pengertian Git
		Yaitu sebuah Version Control yang memungkinkan masing-masing programmer bisa bekerjasama dalam team meski ada perbedaan jarak dan waktu. Setiap revisi yang dibuat akan ada history nya.

b.	fungsi beberapa command:
		- git init ==> sebagai inisiasi pada directory tersebut untuk dilakukan integrasi ke git
		- git remote add origin <link> ==> memberikan koneksi kepada link di Gitlab / Github
		- git add <file name> ==> menambahkan file yang akan kita upload pada Git
		- git commit -m “<message-name>” ==> memberikan comment/message pada file
		- git push origin master ==> meng-upload file yang sudah diberikan message sebelumnya.
